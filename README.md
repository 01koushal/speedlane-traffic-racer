# 🚗 SpeedLane – 2D Traffic Racing Game

SpeedLane is a **fast-paced 2D traffic racing game built using Python and Pygame** where the player must dodge incoming traffic and survive as long as possible.

The game gets progressively harder as the speed increases, making it a fun reflex-based challenge.

---

## 🎮 Gameplay

* Control your car and avoid incoming traffic
* The longer you survive, the faster the traffic becomes
* Try to beat your **highest score**

---

## ✨ Features

* Smooth **2D arcade-style gameplay**
* **Dynamic difficulty** (speed increases over time)
* **Score and High Score tracking**
* **Pause and resume functionality**
* **Collision detection system**
* **Scrolling background road animation**
* **Game over screen with restart option**
* **Sound effects and background music**

---

## 🕹 Controls

| Key           | Action               |
| ------------- | -------------------- |
| ⬅ Left Arrow  | Move Left            |
| ➡ Right Arrow | Move Right           |
| Enter         | Start / Restart Game |
| P             | Pause Game           |
| R             | Resume Game          |

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/speedlane-traffic-racer.git
cd speedlane-traffic-racer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the game:

```bash
python app.py
```
